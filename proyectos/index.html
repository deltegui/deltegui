Proyectos

Quartz

Quartz es un lenguaje de programación interpretado y con tipado estático. Tanto el mismo compilador/intérprete como su stdlib están escritos en C y Quartz. Por el momento, como esta en fases tempranas de desarrollo, el código no esta disponible.

TeMSYS

TeMSYS es un pequeño sistema que se encarga de recopilar información de sensores (normalmente arduinos o similares) y representa la información con gráficas. El sistema lleva funcionando desde el año 2019 en una Raspberry PI. Al ser un sistema sencillo, me ha permitido implementarlo en distintos lenguajes para probarlos. Ha pasado por Java + Spring, PHP, Node.js (con microservicios) y, la versión actual, Go. El backend usa una implementación de la “clean architecture” de uncle bob, además de la organización de código de un proyecto en Go propuesta por Ben Johnson. El código esta en GitHub.

Plantio

Plantio es un jueguito web que va sobre cultivar plantas en tiempo real. A las plantas les afecta el clima del sitio donde estés físicamente (por ejemplo, si donde estás está lloviendo, no hace falta que las riegues). Le falta por pulir algunas cosas (como algunos comandos que se usan para interactuar con las plantas) e implementar otras (que tu cuenta se borre si pierdes), pero la idea general está terminada. El backend esta hecho en Java + Spring, y, al igual que TeMSYS, usa una implementación de la “clean architecture” de uncle bob. El frontend usa Vue.JS. Lo puedes encontrar en GitHub.

TIS-80

TIS80 es un emulador de una máquina ficticia. Es lo que llaman un “fantasy computer” o, si su cometido son los videjuegos, “fantasy console”. Su nombre es una fusión entre el videojuego TIS 100 y la fantasy console TIC-80. Fue un reto de dos semanas, así que le faltan algunas cosas. La puedes programar en un ensamblador propio. El programa que te permite ensamblar y desenamblar el código esta escrito en Go, mientras que la máquina virtual esta en C. Su código está en GitHub.

configloader

ConfigLoader es una pequeña librería para cargar la configuración desde un fichero JSON, variables de entorno o parámetros de la línea de comandos en una estructura de Go. Casi cualquier proyecto que hago en Go tiene esta librería. GitHub.

SimpleJsonValidator

Una librería para NodeJS y express que valida los JSON de una API rest. Como ya no suelo usar JS para backend, practicamente no la uso. GitHub.
